# 🎬 Gestor de Series de TV

Aplicación **full-stack** para gestionar series de televisión y sus temporadas.  
Permite crear, editar, eliminar y filtrar temporadas pertenecientes a distintas series y plataformas de streaming.

---

# 🚀 Características

## 📺 Gestión de Series
- Listado de series disponibles
- Asociación de temporadas a cada serie
- Orden alfabético automático

## 🎞 Gestión de Temporadas
- Crear nuevas temporadas
- Editar temporadas existentes
- Eliminar temporadas
- Validaciones de reglas de negocio

## 🔎 Filtros y Búsqueda
- Filtrar por **título de la serie**
- Filtrar por **plataforma**
- Filtrar por **género**
